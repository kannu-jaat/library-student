# Library-Student App

## Project Overview
Library-Student app Flutter mein banaya gaya hai jo students ko library books, notices, attendance, aur fees ka management provide karta hai.

## Features
- 📚 Library Management
- 📢 Notices
- 📋 Attendance Tracking
- 💰 Fees Management
- 👤 Student Profile
- 🔐 Authentication

## Getting Started
Flutter SDK install karke `flutter pub get` run karo.
